# CodeAlpha Web Scraping, EDA & Visualization Project

## Overview
This project was completed as part of the Data Analytics Internship at CodeAlpha.

The project includes:
- Web Scraping
- Exploratory Data Analysis (EDA)
- Data Visualization

## Technologies Used
- Python
- BeautifulSoup
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
1. Extracted data from websites using BeautifulSoup.
2. Cleaned and analyzed the dataset.
3. Created visualizations and extracted insights.

## Key Insights
- Trend analysis
- Correlation analysis
- Distribution analysis
